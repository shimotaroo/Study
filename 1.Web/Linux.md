- 権限変更
```
$ sudo chmod 755 {ディレクトリ名 or ファイル名}
```

- ディレクトリ、ファイルの所有者を確認
```
$ ls -l
```

- ディレクトリ、ファイルの所有者を変更
```
$ sudo chown -R {所有者名}:{グループ名} {ディレクトリ名 or ファイル名}
```
