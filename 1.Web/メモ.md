# curl
- 任意のサーバーやWebサイトへhttpリクエストを送って、そのレスポンスをチェックすることができるコマンド
- HTTPのGETやPOSTはもちろん、HTTPSやFTP, Telnet, SMTP, IMAP, POP3など様々な通信プロトコルをサポート
